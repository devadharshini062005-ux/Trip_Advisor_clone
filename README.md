# 🦉 Tripadvisor UI Clone

A beginner-friendly clone of the [Tripadvisor](https://www.tripadvisor.com) website built using only **HTML** and **CSS**

---

## 📁 Project Structure

```
tripadvisor-clone/
│
├── index.html
├── Style.css
└── README.md
   
```

---

## ✨ Features

- ✅ Sticky-style Navbar with logo, Plan with AI button, and Sign In
- ✅ Hero section with "Where to?" heading
- ✅ Search bar with tab filters (Hotels, Things to Do, Restaurants)
- ✅ Secondary navigation bar
- ✅ "Find things to do by interest" card grid
- ✅ "Must-dos in amazing places" experience cards with ratings
- ✅ "Iconic places you need to see" destination cards
- ✅ Full footer with links, columns, and copyright

---

## 🛠️ Built With

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure and content |
| CSS3 | Styling, layout, colors, gradients |

> **No JavaScript. No frameworks. No external libraries.**

---

## 🎯 Who Is This For?

This project is perfect for:

- 👶 **Beginners** learning HTML and CSS
- 🎨 **Frontend learners** who want to practice layout with Flexbox
- 📚 **Students** doing UI clone assignments
- 💼 **Portfolio builders** looking for a clean beginner project

---

## 📖 Concepts You Will Learn

- HTML page structure (`div`, `p`, `a`, `input`, `button`)
- CSS Flexbox for layouts (`display: flex`, `gap`, `flex: 1`)
- CSS gradients (`linear-gradient`)
- Hover effects (`:hover`)
- Positioning (`position: relative`, `position: absolute`)
- Border radius and rounded buttons
- CSS comments and clean code organization
